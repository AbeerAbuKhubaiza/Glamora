# Glamora – Beauty Booking App

Glamora is a mobile beauty booking application built using Flutter and Firebase.  
The app allows users to discover beauty salons, browse services, book appointments, and manage their bookings efficiently.

---

##  Features

### User Side
- Browse salons and beauty services
- View service details, prices, and durations
- Book appointments easily
- Track booking status (pending, accepted, completed, cancelled)
- Manage user profile and favorites
- Leave reviews and ratings

### Salon Owner Side
- Manage salon services
- View and manage bookings
- Update booking and payment status
- Track salon performance through dashboard data

---

##  Technologies Used

- *Flutter*
- *Firebase Realtime Database*
- *Firebase Authentication*
- *GitHub*
- *Clean Architecture principles*

---

##  App Structure

The application is divided into two main sections:
- *User Application* – for clients booking beauty services
- *Owner Dashboard* – for salon owners to manage bookings and services

Both sections are integrated with a scalable Firebase backend.

---

##  Purpose

This project was developed to demonstrate:
- Mobile app development with Flutter
- Real-time data handling using Firebase
- Clean and scalable application structure
- Practical booking system implementation
